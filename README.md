less-style-guide
================

Style guide for projects that use LESS

####Naming Conventions
Class names should be kept short and use dashes to seperate words

```
.editor-tabs {
	height: 135px;
	list-style-type: none;
	margin-left: auto;
	width: 95%;
}

.tile-container {
	margin: 0 auto;
	text-align: center;
	width: 95%;
}
```
